Sobre o Projeto

FateNotes

Projeto desenvolvido, para a matéria de Laboratório de Desenvolvimento de Banco de Dados 3, da FATEC - SJC, o projeto tem como necessidade o uso de uma ferramenta de automação de compilação em JAVA, neste projeto está sendo utilizado Maven.

A ferramenta FateNotes, tem como objetivo proporcionar que o usuário possa criar e deletar rapidamente, anotações ou textos sobre o assunto que quiser.
